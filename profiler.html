HOW TO GENERATE A PROFILE FILE
1
Build NNTrainer with profiling on your Ubuntu server: cd /storage_data/Snap/nntrainer meson setup build --buildtype=release -Db_cpp_args="-DPROFILE" ninja -C build -j$(nproc)
2
Run your model and save output to a file: ./build/Applications/mnist/mnist 2>&1 | tee mnist_profile.txt
3
Copy the file to your Windows PC: scp test@107.99.46.57:~/nntrainer/mnist_profile.txt C:\Users\You\Downloads\
4
Click Load Profile and pick the .txt file.
